# Art Portfolio

Welcome to my art portfolio! This project showcases my artwork and provides information about me as an artist. Below are the details on how to navigate and set up the portfolio.

## Project Structure

The project is organized as follows:

```
art-portfolio
├── index.html          # Main entry point for the portfolio
├── css
│   ├── main.css       # Custom styles for the portfolio
│   └── normalize.css   # CSS reset for consistent styling
├── js
│   └── main.js        # JavaScript for interactive elements
├── assets
│   ├── artwork        # Directory for artwork files
│   └── fonts          # Directory for font files
├── pages
│   ├── about.html     # About page for artist information
│   ├── contact.html   # Contact page with form or details
│   └── gallery.html   # Gallery page showcasing artwork
├── .gitignore         # Files and directories to ignore by Git
├── LICENSE            # Licensing information for the project
└── README.md          # Documentation for the project
```

## Getting Started

To view the portfolio, open the `home.html` file in your web browser. This file serves as the main entry point and links to all necessary CSS and JavaScript files.

### Prerequisites

Make sure you have a modern web browser to view the portfolio correctly.

### AI Protection (Glaze)

This project uses [Glaze](https://glaze.cs.uchicago.edu/) to protect artwork from AI mimicry.
Please refer to [GLAZE_WORKFLOW.md](../GLAZE_WORKFLOW.md) for instructions on how to process new artwork before adding it to the site.

### Installation

1. Clone the repository to your local machine:
   ```
   git clone https://github.com/yourusername/art-portfolio.git
   ```
2. Navigate to the project directory:
   ```
   cd art-portfolio
   ```
3. Open `index.html` in your preferred web browser.
